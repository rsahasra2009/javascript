📖 Top 15 Git Commands
# Command What It Does Example & Notes
1 git init Create a new repository mkdir my-project && cd my-project git init Initialized empty Git repository in .../.git/
2 git status Check file status (tracked / modified / staged) echo "Hello" > hello.txt git status 💡 Run this constantly — it's your best friend.
3 git add Stage files for commit git add hello.txt # single file git add . # all files git add *.txt # by extension
4 git commit Save staged changes as a snapshot git commit -m "Add hello.txt" git commit -am "Update files" 💡 Write meaningful messages!
5 git log View commit history git log --oneline git log --oneline --graph --all
6 git diff See line-by-line changes git diff # unstaged git diff --staged # staged 💡 Review before committing.
7 git branch Create / list / delete branches git branch feature-login # create git branch -d feature-login # delete
8 git switch Switch branches (modern) git switch feature-login # switch git switch -c feature-signup # create+switch Also works: git checkout -b
9 git merge Combine branches git checkout main git merge feature-login 💡 Switch to destination first, then merge.
10 git remote Link to GitHub git remote add origin https://github.com/USER/REPO.git git remote -v
11 git push Upload commits to remote git push -u origin main # first time git push # after that
12 git pull Download & merge from remote git pull git pull origin main 💡 Always pull before you push!
13 git clone Copy a remote repository git clone https://github.com/USER/REPO.git git clone -b branch https://github.com/USER/REPO.git
14 git stash Temporarily save work git stash # save git stash pop # restore git stash list # list all
15 git reset Undo changes git reset hello.txt # unstage git reset --soft HEAD~1 # undo commit, keep changes git reset --hard HEAD~1 # undo commit, DELETE changes ⚠️ --hard is dangerous! Use --soft to stay safe.
